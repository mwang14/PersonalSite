Projects

Last year, I decided to make a multiple choice test maker in Java. The user would input questions through a Swing UI, and then when finished, the program would make word documents for multiple versions of the test the user gave, and answer keys for each version. I plan to use my new knowledge of Django to remake this program as a web app with updated functionality including the ability to save tests for future work.



I started one of my first major projects when I learned about the Taylor Series in high school. It was one of the first times that I saw a real application of the math I was learning in class.
